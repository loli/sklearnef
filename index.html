sklearnef¶

Release:0.1.dev
Date:September 18, 2017

sklearnef is an extension library for sklearn, providing density and semi-supervised forests written in Python.

sklearn.ensemble: Ensemble methods (decision forests)¶

The sklearnef.ensemble module includes ensemble-based methods for density learning and semi-supervised classification.

ensemble.DensityForest([n_estimators, ...]) A forest based density estimator.
ensemble.SemiSupervisedRandomForestClassifier([...]) A forest based semi-supervised classifier.

sklearn.tree: Decision trees¶

The sklearnef.tree module includes decision tree-based models for density learning and semi-supervised classification, as well as some density classes for further operations on density trees.

tree.DensityTree([criterion, splitter, ...]) A tree for density estimation.
tree.SemiSupervisedDecisionTreeClassifier([...]) A tree for semi-supervised classification.
tree.GoodnessOfFit(cdf, X)
tree.MECDF(X)
